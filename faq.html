Important FAQ

What is the Document Object Model (DOM)?

Document Object Model is a object representation of the HTML document. This DOM represents elements in the page such that they be accessed and modified.

How do you select and element from the DOM?

We select elements from the DOM by using javascript methods and the elements' ids, class names etc.

What is event delegation in the context of the DOM, and why it is useful?

Events in a element bubble up the DOM tree. Event delegation is delegating or using the parent element to target the element instead of using the element itself. This helps us write cleaner code since do not have to listen to the event on every child element rather just listen to the parent
